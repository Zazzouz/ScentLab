# Custom Fragrance Builder

## Project Description
Custom Fragrance Builder is a web application that allows users to design their own dream fragrance. Users can select different fragrance notes and combine them into a structured perfume blend using top, middle, and base notes. The goal of the project is to create a simple interactive tool that demonstrates how fragrances are built and how different scent notes work together.

## Planned Features

1. **Fragrance Builder**  
   Users will be able to create a custom fragrance by selecting top, middle, and base notes from a list of common fragrance ingredients.

2. **Fragrance Pyramid Visualization**  
   The application will visually display the fragrance structure using the traditional perfume pyramid (top notes, heart notes, and base notes).

3. **Save Your Fragrance**  
   Users can give their fragrance a custom name and save their scent combinations so they can revisit them later.

4. **Fragrance Note Library**  
   A page that explains common fragrance notes such as bergamot, sandalwood, amber, and vanilla, helping users understand the characteristics of each ingredient.

5. **Blend Strength Controls**  
   Users will be able to adjust the intensity of each fragrance layer using simple sliders to simulate the balance of a real perfume blend.

## Tech Stack
- **HTML** – structure of the website  
- **CSS** – styling and layout  
- **JavaScript** – interactivity and fragrance builder functionality  
- **Cursor AI** – development assistance  
- **Netlify** – site deployment
